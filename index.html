Introduction

This is a demonstration of a basic web page created with semantic HTML and styled using an external CSS file. The goal is to showcase the fundamentals of web page structure and styling, including the **CSS Box Model** and various **CSS selectors**.

The Box Model Explained

Every HTML element is a box. The CSS Box Model consists of **margin**, **border**, **padding**, and the content itself. This structure helps us control the spacing and size of elements on a page.

Consistent Layout

By using CSS rules, we can achieve a consistent layout across different elements. This includes uniform sizing, spacing, and alignment, which creates a professional and easy-to-read user interface.

Proper Selector Use

CSS selectors allow us to target specific elements to style. This file uses a combination of element selectors (`body`, `h1`), class selectors (`.card`, `.intro-section`), and ID selectors (`#footer-info`) to apply styling efficiently.
